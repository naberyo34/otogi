# otogi
## 概要
- オンラインでのTRPGを楽しむツールです。
- 徐々に機能追加していく、かも。
- GMの進行の容易さを最優先に考え、まずは最低限の確実に役立つ機能のみに絞って開発していきます。
## 機能
- リアルタイムダイスロール
## 開発コマンド
- `yarn start`でプレビュー起動。
- `yarn build`で`build`フォルダへのビルド。
- `firebase deploy --only hosting`で`build`フォルダをデプロイ。
## 技術
- TypeScript
- React (Hooks)
- Redux, React-Redux
- Firebase
